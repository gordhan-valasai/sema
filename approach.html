Working Approach

Six-step process for optimisation-based analysis

1

Problem Definition

Clarify decision variables, objectives, constraints, and uncertainty sources. Establish scope boundaries and identify key assumptions.

2

Data Collection and Processing

Gather historical data, technology parameters, and scenario inputs. Process time-series data (hourly renewable profiles, demand, prices) and validate data quality.

3

Model Formulation

Develop mathematical formulation in LaTeX notation. Select appropriate optimisation paradigm (MILP, stochastic, agent-based) and framework (Pyomo, JuMP, TIMES, OSeMOSYS).

4

Implementation and Solving

Implement model in chosen framework. Select solver (Gurobi, CPLEX, HiGHS) based on problem structure. Validate model correctness and check solution feasibility.

5

Sensitivity Analysis

Test sensitivity to key parameters (costs, demand, policy constraints). For stochastic models, calculate value of stochastic solution and expected value of perfect information.

6

Documentation and Reporting

Document model assumptions, constraints, and limitations. Create publication-quality visualisations. Prepare technical report with reproducible code repository.

Documentation Standard: All models are documented with assumption registers, validation checks, and sensitivity analyses to support independent review.

Principles

Transparency

All assumptions explicitly stated and documented

Reproducibility

Code and data available for independent verification

Rigour

Mathematical formulations validated and tested

Clarity on Limitations

Model boundaries and uncertainties clearly communicated
